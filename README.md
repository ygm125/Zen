# Zen

**Web framework based on KOA**

## Install
```js
git clone https://github.com/ygm125/Zen.git
npm install // 安装依赖
gulp server // 启动node服务
gulp dev  // 开发环境监听编译
gulp deploy // 编译并部署
```
## Introduction
- js支持es6语法，前端js模块化书写，自动合并打包，无需加载器
- image自动压缩
- css采用less编写
- 引用资源md5更新
- 资源修改自动reload
