# Zen

**Web framework based on KOA**

## Install
```js
git clone https://github.com/ygm125/Zen.git
npm install // 安装依赖
gulp init //首次执行
gulp server // 启动node服务
gulp dev  // 开发环境监听编译
gulp deploy // 编译并部署
```
## Introduction
- src为需要编译的源码目录
- controller使用es6语法
- js同样使用es6语法，模块化书写，自动合并打包，无需加载器
- image自动压缩
- css采用less编写
- 引用资源md5更新
- 资源修改自动reload
